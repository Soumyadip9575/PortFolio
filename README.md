# 🚀 PortFolio

A modern and responsive personal portfolio website developed using **Flask, HTML, CSS, and JavaScript**.

The platform showcases my professional profile, AI/ML skills, academic achievements, projects, certifications, downloadable resume, and contact information in a clean and interactive interface.

🌐 **Live Demo:** https://portfolio-24ff.onrender.com

---

## ✨ Key Features

- 🎨 Modern Dark-Themed UI Design
- 📱 Fully Responsive Layout
- 👨‍💻 Professional Portfolio Showcase
- 📂 Dynamic Project Gallery
- 🏆 Certification & Achievement Section
- 📄 Downloadable Resume / CV
- 📞 Contact Form Integration
- 🔗 Social Media Integration
- 🌙 Dark / Light Theme Toggle
- ⚡ Smooth Scrolling Navigation
- 🚀 Flask Backend Support
- ☁️ Render Cloud Deployment
- 🔄 GitHub Version Control
- 📈 Optimized Performance
